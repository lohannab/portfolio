## 📖 Sobre o Projeto

Portfólio pessoal desenvolvido com **HTML, CSS e JavaScript puros**, sem frameworks ou dependências externas.

O objetivo é apresentar minha trajetória profissional e evolução como desenvolvedora **Full Stack em formação** — com uma identidade visual minimalista, paleta editorial em preto, creme e dourado.

---

## 🎨 Design

| Atributo | Detalhe |
|----------|---------|
| **Estilo** | Minimalista editorial |
| **Paleta** | Preto, creme e dourado |
| **Abordagem** | Mobile-first |
| **Foco** | Hierarquia visual e clareza |

---

## ✨ Funcionalidades

| Recurso | Descrição |
|---------|-----------|
| 📱 **Responsividade** | Layout mobile-first adaptado para todos os tamanhos |
| 🎞️ **Animações** | Elementos animados com `IntersectionObserver` |
| 🔲 **Navbar** | Efeito de blur ao rolar a página |
| 🃏 **Cards interativos** | Efeito hover nos projetos |
| 🖤 **UI limpa** | Estética minimalista e sem distrações |

---

## 📄 Páginas

### 🏠 Home (`index.html`)
- Hero de apresentação
- Seção sobre mim
- Grid de projetos
- Links de contato

### 📬 Contato (`contato.html`)
- Formulário com nome, e-mail, assunto e mensagem
- Feedback visual de envio

---

## 🧱 Estrutura do Projeto

```
portfolio/
│
├── index.html        # Página principal
├── contato.html      # Página de contato
├── style.css         # Estilos globais
└── lohannab.jpg      # Imagem de perfil
```

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|------------|-----|
| **HTML5** | Estrutura semântica das páginas |
| **CSS3** | Flexbox, Grid e variáveis CSS |
| **JavaScript Vanilla** | Animações e interações |

> Sem frameworks. Sem dependências. Zero configuração.
